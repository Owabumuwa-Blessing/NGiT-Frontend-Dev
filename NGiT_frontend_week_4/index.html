Weekly Reflection

Understanding Computers, Operating Systems, and the Web

In my first week as a frontend developer, I learned how computers, operating systems, and the web work. Understanding how the CPU, memory, and file system operate helps me write efficient, secure, and reliable code. Learning how browsers, servers, and the internet interact makes debugging easier and improves application performance. These fundamentals provide a strong foundation, making it easier to build scalable, reliable, and well-optimized websites.

How the Web Works & HTML Basics

I learned how the web works: the browser requests HTML from the server, which is then displayed in the browser. I practiced building pages with headings, paragraphs, lists, links, and images, understanding the difference between block and inline elements.

Forms, GET vs POST, and Media Embeds

I learned how to create HTML forms and the difference between GET (for retrieving data) and POST (for sending data securely). I also embedded images, videos and audio, making web pages interactive and engaging for users.

Exploring CSS: Inline & Internal Styling

I began styling elements using inline and internal CSS. I practiced formatting lists, images, and page sections, gaining confidence in organizing my content semantically. Applying borders, colours, and mastering semantics HTML made my page more visually appealing.
