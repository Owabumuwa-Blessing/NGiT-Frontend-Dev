Month 1 Reflection

Understanding Computers, Operating Systems, and the Web

In my first week as a frontend developer, I learned how computers, operating systems, and the web work. Understanding how the CPU, memory, and file system operate helps me write efficient, secure, and reliable code. These fundamentals provide a strong foundation for building scalable websites.

How the Web Works & HTML Basics

I learned how the browser requests HTML from the server and displays it. I practiced building pages with headings, paragraphs, lists, links, and images.

Forms, GET vs POST, and Media Embeds

I learned how to create HTML forms and the difference between GET and POST methods. I also embedded images, videos, and audio to make web pages interactive.

Exploring CSS: Inline & Internal Styling

I practiced styling elements using CSS, applying borders, colours, and layouts to improve the visual structure of my pages.

I’m Blessing Owabumuwa, an aspiring Frontend Developer and a member of the first cohort of Naija Girls in Tech (NGiT). I am a creative problem-solver transitioning into web development, aiming to create intuitive, visually appealing, and user-friendly digital experiences.
